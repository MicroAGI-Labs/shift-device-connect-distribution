iOS Installation Guide

  1. Tap the "Download for iOS" button on your iPhone
  2. If TestFlight isn't installed, you'll be redirected to the App Store — install it first
  3. Return to this page and tap the download button again
  4. Tap "Accept" to join the beta testing program
  5. Tap "Install" to download Shift Device Connect

Android Installation Guide

  1. Tap the "Download for Android" button to download the APK file
  2. Open the downloaded file from your notifications or file manager
  3. If prompted, allow installation from unknown sources: Go to Settings > Security (or Apps & notifications) and enable "Install unknown apps" for your browser
  4. Tap "Install" and wait for the installation to complete
  5. Open Shift Device Connect from your app drawer
